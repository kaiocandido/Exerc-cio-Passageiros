# CALCULA A QUANTIDADE DE LOTAÇÃO EM UMA SALA


# Logica
- Pegamos a quantidade de passageiros
- Armazenatos através de nome,tempo de entrada e tempo de saida
- Colocamos os valores em tres listas, nomes, e (entrada), s(saida)
- Foi criado um for onde lemos as informações nome, e(entrada, s(saida) e adicionamos nas listas
- Usamos uma lista chamada delta para armazenar o tempo limite sendo de 0 até 1000
- Criamos um laço for para salvar dentro da lista delta a saida e entrada de cada passageiro
- Epor ultimo criamos um laço para ver a quantidade de passageiros na sala

# 1- EXEMPLO

Pessoa A: 1–5
Pessoa B: 5–10
Resultado: 1

# 2- EXEMPLO

Pessoa A: 1–5
Pessoa B: 2–10
Resultado: 2
